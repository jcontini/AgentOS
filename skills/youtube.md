# YouTube Skill

## Intention: Handle YouTube links (transcript or video download)

When the user provides a YouTube link and asks for transcript or video download:

### Actions

- **Transcript only:** `/Users/joe/dev/ai/scripts/youtube-transcript.sh "[YOUTUBE_URL]"`
- **Video + transcript:** `/Users/joe/dev/ai/scripts/youtube-transcript.sh "[YOUTUBE_URL]" --video`

### Output locations

- Transcripts: `/Users/joe/dev/ai/user/youtube/transcripts/`
- Videos: `/Users/joe/dev/ai/user/youtube/videos/`
